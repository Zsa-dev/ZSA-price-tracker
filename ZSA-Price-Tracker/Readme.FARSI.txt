# 💰 ZSA Price Tracker

اپلیکیشن دسکتاپ برای دریافت قیمت لحظه‌ای:
- 🔸 طلا
- 🔸 بیت‌کوین و سولانا
- 🔸 زعفران


## 📸 اسکرین‌شات‌ها
<img src="screenshots/app.png" width="400"/>

## ✨ ویژگی‌ها
- رابط کاربری ساده، تاریک و مدرن
- دریافت داده از منابع زنده آنلاین
- قابل اجرا روی ویندوز 
- کدنویسی با Python و Tkinter 

## 📦 فایل‌ها
- `main.py` → کد اصلی
- `prices_history.csv` → تاریخچه قیمت‌ها
